URL Shortener Microservice

By: Josh Sanders

This microservice is used to shorten urls.

Examples of how to use this service:

To setup a URL: https://mins.herokuapp.com/new/http://foobar.com

(This will return: { "original_url":"http://foobar.com", "short_url":"https://mins.herokuapp.com/10" } )


Then use the value supplied as "short_url": https://mins.herokuapp.com/10
