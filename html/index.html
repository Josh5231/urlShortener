URL Shortener Microservice

By: Josh Sanders

This microservice is used to shorten urls.

Examples of how to use this service:

To setup a URL: https://url-shortener-josh5231.c9users.io/new/http://foobar.com

(This will return: { "original_url":"http://foobar.com", "short_url":"https://little-url.herokuapp.com/8170" } )


Then use the value supplied as "short_url": https://little-url.herokuapp.com/8170
